Réalisation d'une Base de données

Compétence : Gérer des données de l'information

Résumé

Le diagramme de classe nous guide dans la création des différentes classes nécessaires et dans la définition des multiplicités pour chaque table. Nous avons élaboré ce diagramme en amont, ainsi que la mise en place des données de la base de données sous SQL. En nous basant sur ce diagramme, nous avons conçu la structure de notre base de données pour refléter efficacement notre modèle conceptuel. Cette phase de conception représente une étape cruciale de notre projet, permettant de poser des bases solides pour le développement ultérieur.

Compétences Apprises ou Améliorées

Compétences : Modélisatin UML, Compréhension des bases de donées, Analyse

Langage(s) : SQL
